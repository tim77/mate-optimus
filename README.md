## Building, Testing, and Installation

### Ubuntu

```bash
sudo apt install mate-optimus
```

### Fedora

```bash
sudo dnf install mate-optimus
```

### Source

You'll need the following dependencies:

  * `gir1.2-gtk-3.0`
  * `gir1.2-notify-0.7`
  * `glew-utils`
  * `python3`
  * `python3-gi`
  * `python3-setproctitle`
